# Prompts photos — Soly'bat

18 prompts prêts à copier-coller dans un générateur d'image IA (Midjourney, DALL-E, Stable Diffusion...). Chaque prompt est autonome : pas besoin d'y ajouter quoi que ce soit.

Format cible : paysage 4:3.

**Important : Soly'bat ne fait pas de rénovation extérieure** (pas de façade, toiture, ravalement). Tous les prompts doivent rester en intérieur — pas de photo de façade ou de chantier vu depuis la rue.

---

## Après génération

Dépose les images dans `src/images/realisations/` (ou `src/images/` pour les heros), puis :
- Heros : renseigne `image:` dans le front-matter de `index.njk`, `renovation-location.njk`, `realisations/index.njk`, `espace-pro.njk`.
- Réalisations : renseigne `photos.avant`/`photos.apres` dans [realisations.json](../src/_data/realisations.json).

Les composants basculent automatiquement du placeholder vers l'image dès qu'un chemin est renseigné.
